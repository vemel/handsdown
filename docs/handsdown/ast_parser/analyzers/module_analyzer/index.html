    ModuleAnalyzer().visit_Assign

    Show source in module_analyzer.py:127

    Parse info about module attribute statements.

    Adds new ast.Assign entry to attribute_nodes. Skips assignments to anything pther that a new variable. Skips multiple assignments. Skips assignments with names starting with _. Parses __all__ and add all values to all_names

    Examples

    MY_MODULE_ATTR = 'value'
    my_attr = "value"
__all__ = ['MyClass', 'my_func']

# these entries are skipped
_MY_MODULE_ATTR = "value"
multi_attr_1, multi_attr_2 = [1, 2]
my_object.name = "value"
__all__ = all_list
